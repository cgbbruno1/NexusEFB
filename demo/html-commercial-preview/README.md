# NexusEFB Offline HTML Commercial Preview

Este diretorio contem um prototipo comercial HTML offline do NexusEFB.

## Objetivo

Fornecer uma apresentacao visual local, simples e independente do app Flutter para demonstracao comercial do conceito do MVP.

## Escopo

- arquivo unico `index.html`
- abre direto no navegador
- nao depende de internet
- nao depende de Flutter
- nao altera o app real
- nao integra com `lib/`, `android/` ou `ios/`

## Modulos demonstrativos presentes

- Moving Map
- Cartas Offline
- Plano de Voo
- METAR / TAF / NOTAM
- Logbook Basico
- Cadastro de Aeronaves

## Como abrir

1. Abrir `demo/html-commercial-preview/index.html` direto no navegador.
2. Navegar pelas abas da demo.

## Limitacoes

- prototipo comercial nao operacional
- sem dados reais de voo, cartas, meteorologia ou manutencao
- sem conectividade com APIs
- sem persistencia de dados
- sem qualquer vinculacao com o app Flutter de producao
- nao substitui EFB certificado
- nao implementa funcoes IFR certificadas

## Observacao importante

Este material serve apenas para apresentacao visual e alinhamento comercial. Nao deve ser utilizado para navegacao, briefing operacional, despacho, manutencao ou qualquer decisao aeronautica real.
